# What is Shell?
1. Shell mein hum Django ka Backend ka kam bina Website ka Test kar sekhte hai or Backend ka code run ho reha hai yeah nahe.

2. Shell ko active karne ka liya hum apne `CMD` yeah `Terminal` per yeah command likhte hai.
- `python manage.py shell` -> isa Shell active hojayga

# Shells Commands:

```py
1. form home.models import * # -> home mein models.py ko import kiya hai
2. Patient(name = "Kaif" , age = 21 , email = "shahkaif327@gmail.com") # is command se Patient Model mein name , age or email in column mein aik record add kiya hai
3. Patient.save() # isa yeah record save hojayga
```
