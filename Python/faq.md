# What is REPL in Python?
1. `REPL` ko Activate karne ka kiya hum terminal per `py` , `python3` yeah `python` yeah likhna hota ha
2. Code ko Quick dekhna oisa run karna Result Quick dekhna Test aur practice karna.

| REPL                  | .py file              |
| --------------------- | --------------------- |
| R  -> | Read (tumhara code read karta hai)           |
| E  -> | Evaluate (usko run karta hai)                |
| P  -> | Print (result dikhata hai)                   |
| L  -> | Loop (phir se next input ka wait karta hai)  |
