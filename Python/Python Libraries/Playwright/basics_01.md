# What is Playwright:
1. Playwright ek modern web automation framework hai jo Microsoft ne 2020 mein banaya tha . Iska kaam hai web browsers ko automate karna, jaise websites kholna, buttons click karna, forms bharna, tests run karna, aur data extract karna

2. Playwright ek open-source tool hai jo aapko code likh kar browser control karne ki facility deta hai. Aap bina manually browser operate kiye, automated tasks kar sakte ho

3. Playwright yeah Python ki Library or Framework dono hai.

# Main Purpose of PlayWright:
1. Playwright teen **main kaamo** ke liye use hota hai :

| Purpose | Explanation |
|---------|-------------|
| **Web Testing / QA** | Websites ki testing automate karna — check karna ke sab buttons, links, forms sahi kaam kar rahe hain |
| **Web Scraping** | Websites se data nikalna (products, prices, news, etc.) chahe wo JavaScript se dynamic load hota ho |
| **RPA (Robotic Process Automation)** | Repetitive web tasks automate karna jaise forms fill karna, reports download karna |

# Cross-Platform Support PlayWright:

**Cross-platform** ka matlab hai ki ek **same code/tool** different operating systems (OS) par **bina change kiye** kaam karta hai.

**Simple example:**
- Aapne **Windows** computer par ek automation script likha
- Wohi script **Mac** par bhi chalega
- Wohi script **Linux** (Ubuntu, etc.) par bhi chalega
- **Code mein koi change nahi karna padega**

Iske opposite hai **platform-specific** tools jo sirf ek OS par chalte hain (jaise .NET Framework pehle sirf Windows par chalta tha).
