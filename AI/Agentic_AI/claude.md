1. first download Claude Code in globally run this cmd `npm install -g @anthropic-ai/claude-code` yeah `npm` package manager se download hoga or npm package manager nodejs ke sath aata hai

2. cmd mein run karna hai `claude` tu yeah kch question puchega jaise ka:

    a. `Dark mode` -> enter <br>
    b. ` Anthropic Console account · API usage billing` -> enter <br>
    c. `url ayga oisa copy krna hai` or browser mein paste krna hai or -> enter <br>
    d. oiska bad apne local model ko run karna hai `ollama launch claude --model <model_name>` -> enter -> isme sirf wahi model use hoga jo `Agentic AI` mein use hota hai jinke pass call function capability hai
