# What is Big O Notation:
1. **Big O Notation** computer science aik mathematical tool hai.
2. jo hamain yeh batati hai ke jab hamare program ka input (data) barhega, to hamare code ki performance (chalne ki speed aur memory) par kya asar parega or **algorithm** ka **Execution Time** per kia affect areha hai oisa btati hai.
3. Yeh aap ke code ka exact time (jaise 2 seconds ya 5 mili-seconds) nahi batata, balki yeh **Growth Rate** (barhne ki speed) batata hai.
4. Kyunke aik hi code expensive computer par fast chalega aur saste computer par slow, is liye seconds mein time measure karna ghalat hota hai. **Big O** computer ki speed nahi, balki code ke steps measure karne ka liya use hota hai.
