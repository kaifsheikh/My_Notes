# What is Access Modifier?
1. Access modifier ek rule ya label hai jo decide karta hai ke aapki class ke variables (properties) aur functions (methods) ko kahaan se access (use) kiya ja sakta hai.

| Modifier      | Access                | Example / Easy words                                                                                      |
| ------------- | --------------------- | --------------------------------------------------------------------------------------------------------- |
| **public**    | Anywhere              | Class ke andar bhi, bahar bhi, aur kisi object se bhi access ho sakta hai. So, **sabko dikhai deta hai**. |
| **protected** | Class + child classes | Sirf **apni class aur usse inherit (extend) hui class** use kar sakti hai. Bahar ka code nahi.            |
| **private**   | Class only            | Sirf **apni class ke andar** use hota hai, bahar se ya child class se access nahi.                        |
