# PHP OOP - Easy Overview

## 1. Class & Object
- **Class** = Blueprint / Design  
- **Object** = Real instance of a class  

## 2. Properties & Methods
- **Properties** = Object ke attributes (jaise color, name)  
- **Methods** = Object ke actions (jaise run, eat)  

## 3. Constructor & Destructor
- **Constructor** = Object create hone par run hota hai  
- **Destructor** = Object destroy hone par run hota hai  

## 4. Inheritance
- Ek class apni properties/methods dusri class se le sakti hai  
- Example: `Car` class se `ElectricCar` class inherit kar sakti hai  

## 5. Access Modifiers
- **public** = Sab dekh sakte hain  
- **private** = Sirf class ke andar access  
- **protected** = Class aur inherited class mein access  

## 6. Abstraction
- Sirf important cheez dikhao, unnecessary chhupa do  
- `abstract class` aur `abstract methods` use hote hain  

## 7. Encapsulation
- Data ko safe rakhna (hidden) aur methods ke through access karna  

## 8. Polymorphism
- Same method ka different behavior  
- Method overriding ya interfaces ke zariye  

## 9. Interfaces
- Rules banate hain ki kaunse methods class mein honge  

## 10. Traits
- Multiple classes ke liye common functionality share karna  

## 11. Static Properties & Methods
- Object banaye bina access kar sakte hain  

## 12. Final Keyword
- Class ya method ko inherit ya override nahi kar sakte
