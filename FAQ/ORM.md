# What is ORM?
1. ORM (Object-Relational Mapping) ek aisi Programming technique hai jo do mukhtalif Cheezo ko aapas mein connect karti hai:
2. ORM asal mein ek bridge hai jo `OOP` aur `RDBMS` ko aapas mein jorta hai.
2. `OOP` mein hum Object or Classes per Kam karta hai.
3. `RDBMS` isme hum Tables, Rows, aur Columns per work karte hain.

Concept,OOP (Programming),RDBMS (Database)
Structure,"Class (e.g., Student)","Table (e.g., student_table)"
Data Point,"Property/Variable (e.g., name)","Column (e.g., name_col)"
Single Entry,Object (One instance),Row (One record)

# Type of 2 Things:
1. `(OOP)` Object-Oriented Programming (Python , Java , C#s)
2. `(RDBMS)` Relational Database Management System (MySQL, PostgreSQL, SQLite)
